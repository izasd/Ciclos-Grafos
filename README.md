# Detecção de Ciclos em Grafos Aplicada a Fraude Bancária

Este projeto implementa a identificação de ciclos em grafos a partir de dados de transações financeiras.
O objetivo é mostrar como ciclos podem indicar movimentações suspeitas, frequentemente associadas a fraude ou triangulação.

## Funcionalidades
- Leitura de um grafo a partir de arquivo texto
- Construção do grafo em memória
- Detecção de ciclos simples
- Interface JavaFX para exibir resultados

## Tecnologias
- Java
- JavaFX
- NetBeans

## Arquivo de Entrada
O arquivo `grafo_suspeito.txt` deve listar as transações no formato:  
v1 v2  
v2 v3  
...
